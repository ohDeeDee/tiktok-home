Privacy Policy

Effective Date: November 12, 2025

1. Overview

This Privacy Policy explains how this application (“the App”) handles information when interacting with the TikTok API. The App is created and maintained by a private individual (“the Developer”) and is used strictly for personal and educational purposes.

2. Information Collected

The App does not actively collect, store, or share personal information from TikTok users. Any data received from TikTok (such as webhook events, usernames, or profile statistics) is used only for testing or demonstration purposes, processed temporarily in memory, and not retained permanently.

3. Data Usage

All TikTok data handled by the App is used solely for local testing and non-commercial integration experiments. The data is not saved permanently, shared externally, or used for marketing or analytics.

4. Data Retention

No user data is intentionally stored. Temporary data may exist in memory while the App runs but is deleted once the session ends.

5. Third-Party Services

The App interacts only with TikTok’s official API or webhook endpoints for development purposes. No other third-party services are used.

6. Security

The App is used in a controlled, personal environment. No user data is transmitted beyond TikTok’s API and the developer’s local or hosted server.
